# MD Searcher

一个用于在Markdown文件中快速搜索并定位内容的Java应用程序，支持在Typora等编辑器中精确定位搜索结果。

## 功能特点
- 快速搜索Markdown文件内容
- 自动在默认编辑器中打开文件并定位到搜索位置
- 支持根据文件大小动态调整等待时间
- 针对Typora编辑器优化的定位逻辑

## 系统要求
- Java 8或更高版本
- Windows操作系统
- Typora或其他支持标准键盘快捷键的文本编辑器

## 安装方法
直接下载exe点击即可使用

## 使用说明
1. 在搜索框中输入要查找的文本，点击搜索或按下回车键搜索文本
2. 双击你要搜索的文本，程序会自动打开并定位到
